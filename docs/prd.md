# PG Finder Mohali Requirements Document

## 1. Application Overview

### 1.1 Application Name
PG Finder Mohali

### 1.2 Application Description
A complete multi-page, mobile-responsive web platform for PG (Paying Guest) and Hostel listing directory in Mohali, Punjab, India. The platform enables students to search PG accommodations easily and allows PG owners to list their properties and receive direct enquiries via phone or WhatsApp.

### 1.3 Target Language
English

## 2. Core Features

### 2.1 Home Page
- Large hero banner with search bar
- Search filters: Area (Phase 5, Phase 7, Kharar, Landran, Sector 70, Sector 71), Budget range, Boys/Girls/Co-ed
- Featured PG listings
- Popular locations section
- Recently added PGs
- Why Choose Us section
- Student testimonials
- Call-to-action: Find PG Now

### 2.2 PG Listing Page
- Grid view of multiple PG cards
- Each card displays: Thumbnail photo, PG name, Area, Starting rent, Boys/Girls, Food availability, View Details button

### 2.3 Filter System
Users can filter PGs by:
- Price
- Location
- Gender type
- Food included
- AC/Non-AC
- Attached bathroom

### 2.4 Single PG Detail Page
- Image gallery slider
- Monthly rent
- Security deposit
- Facilities list (WiFi, Laundry, RO Water, CCTV, Power Backup)
- Room types (Single, Double, Triple sharing)
- Availability status (Available/Full)
- Owner contact info
- Call button
- WhatsApp direct message button
- Embedded Google Map
- Schedule Visit button

### 2.5 Area-Wise Pages
Separate pages for:
- PG in Phase 5 Mohali
- PG in Phase 7 Mohali
- PG in Kharar
- PG in Landran
- PG in Sector 70

Each page shows listings only for that specific area.

### 2.6 List Your PG Page (Owner Form)
Form fields:
- Owner Name
- Phone number
- PG name
- Address
- Area selection
- Rent
- Room type
- Facilities checklist
- Photo upload
- Google Map location link

### 2.7 Pricing Plans Section
- Free Trial Listing
- Basic Listing ₹499/month
- Featured Listing ₹999/month (top placement)

### 2.8 Student Features
- Save favourite PG
- Reviews and ratings
- Report fake listing
- Recently viewed listings

### 2.9 Additional Pages
- About Us
- Contact Us
- Privacy Policy
- Terms & Conditions
- FAQ (food timing, security, visitors allowed, curfew time)

### 2.10 Footer
- Phone number
- Email
- Address
- Social media links
